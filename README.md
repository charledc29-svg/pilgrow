# Pigrow 🚀

AI-powered lead generation platform that helps businesses find, qualify, and convert potential customers.

## About

Pigrow uses AI to help businesses discover leads, analyze prospects, automate outreach, and grow faster.

## Features

- AI lead generation
- Lead management dashboard
- AI-powered prospect insights
- Automated outreach
- Analytics
- Team workspace

## Built With

- Wasp
- React
- Node.js
- Prisma
